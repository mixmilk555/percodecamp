2.differentiation tag

Tag ความเหมือน ความต่าง
"< hr> & < br>" เป็น tag ที่ไม่ต้องมี tag ปิด(Empty Elements)
  • hr=เป็นคำสั่งสร้างเส้นคั่น
  • br=เป็นคำสั่งสำหรับขึ้นบรรทัดใหม่
"< ul> & < ol>" เป็นคำสั่งใช้สำหรับสร้างรายการเป็น list
  • ul=สร้งรายการแบบไม่มีลำดับ เน้นการนำเสนอเพื่อสร้างความใจเป็นหลัก
  • ol=สร้างรายการแบบมีการเรียงลำดับเป็นตัวเลข
" < select> & < checkbox> & < radio button> " เป็น tag เกี่ยวกับการเลือกตัวเลือกที่กำหนดไว้ให้เลือก
  • select=เลือกรายการเป็น drop list ได้รายการเดียว
  • checkbox=เลือกรายการได้หลายหลากอันเป็นจุดสี่เหลี่ยมตัวเลือก
  • radio button=เลือกรายการได้แค่อันเดียวโดยเป็นจุดวงกลมตัวเลือก
"< div> & < span>" เป็น tag จัดการเกี่ยวกับการจัดรูปแบบกล่องข้อความเหมือนกัน tag div นั้นจะแยกบรรทัดกล่องข้อความออกจากกล่องข้อความก่อนหน้า ส่วน tag span จะนำกล่องข้อความจะนำกล่องข้อความมาต่อเป็นบรรทัดเดียวกัน
" script & style" เป็นการกำหนดชนิดข้อมูลที่จะใช้ในเอกสารเหมือนกัน script=ทำให้ html สามารถใช้ javascript ในเอกสาร html ได้ style=ใช้กำหนดสไตล์ของข้อมูลเอกสาร html
